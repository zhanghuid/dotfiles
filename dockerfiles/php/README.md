## 构建
```bash
# 构建 8.2 的 fpm 环境
make build php_version=82

# 构建 7.4 的 fpm 环境
make build php_version=74

# 构建 7.2 的 fpm 环境
make build php_version=72

# 构建 7.1 的 fpm 环境
make build php_version=71
```

## 启动
```bash
# 启动 8.2 的 fpm 环境
make run php_version=82

# 启动 7.4 的 fpm 环境
make run php_version=74

# 启动 7.2 的 fpm 环境
make run php_version=72

# 启动 7.1 的 fpm 环境
make run php_version=71
```
